# Running Services

- in the `NJ_COMMUNITY_RESOURCES` folder, run the graphql server:  `npx ts-node src/seed.ts`
- in the `NJ_COMMUNITY_RESOURCES` folder, insert resources.json via seed.ts: `npx ts-node src/seed.ts` 
- in the `community-frontend` folder run the client-side server via vite: `npm run dev`
